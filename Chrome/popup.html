Hide features
Select what you want to hide:
• Auto-play - removes the button from video player controls
• Play on TV - removes the button from video player controls; AKA the Cast button which allows you to cast video to the paired TV
• Join - removes the button from channel actions under the video player
• Clip - removes the button from video actions under the video player
• Thanks - removes the button from video actions under the video player
• YouTube apps - removes the button from the top bar
• Channel avatars on homepage - no effect when 'Extra layout changes' setting is enabled, which already hides channel avatars
• Search with your voice - removes the button to the right of the search bar
• People also search for - removes this section from search results
• Related videos age - removes this info (time since upload date) from related videos
• Video categories - removes this info from video description
• Video licensing - removes this info from video description
• Country code - removes this info from YouTube logo
• Collapsed chat - removes the button to expand chat once it's been collapsed
• Dislikes - removes the dislike button and count from video page
• Download - removes the button from video page
• Shorts - removes the button from homepage and hides shorts in search results
• Chapters in description - removes the chapters section from video description
• Music in description - removes the music section from video description
• Heatmap - removes the heatmap from above the video progress bar
Note: most settings require a page refresh to take effect.
